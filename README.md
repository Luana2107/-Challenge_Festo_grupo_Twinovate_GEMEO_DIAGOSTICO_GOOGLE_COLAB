# -Challenge_Festo_grupo_Twinovate_GEMEO_DIAGOSTICO_GOOGLE_COLAB
Esse é um gêmeo digital diagnóstico com foco na pressão de um atuador pneumático. Esse projeto foi em parceria com a empresa FESTO.

Para rodar, basta abrir o notebook no prório google colab. Em seguida, carregar os dois arquivos .csv presentes no repositório dentro do colab.

OBS: apesar do nome "dados sem pressão", os dados que foram coletados possuem pressão. O nome é devido a um ajuste que foi necessário - retirar um sensor de pressão do nosso sistema de hardware e usar outro.
